## 📌 개요  
**Leeyunseok Portfolio**는 디자이너로서의 정체성과 작업 철학을 담아낸 포트폴리오 웹사이트입니다.  
미니멀한 디자인과 명확한 정보 전달을 중시하며, 사용성과 미적 균형을 함께 고려해 설계되었습니다.
